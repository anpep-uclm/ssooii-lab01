# Lab Assignment 01
> Operating Systems II Course (2020-21)

## Requirements
- A *NIX build environment
- GNU `make`, `gcc` and `g++` with support for C99 and C++14

## Build Instructions
```shell
$ git clone https://github.com/anpep/ssooii-lab01
$ cd ssooii-lab01
$ make all
```
